Fuentes

Elegir una fuente apropiada tiene un fuerte impacto en como los lectores experimentan un sitio web

La fuente correcta puede crear uuna reconocible entidad para tu marca o página web.

Usar una fuente fácil de leer es muy importante. Ya que añade valor a tu texto. También es importante elegir un correcto color y tamaño.

Familias de fuentes genéricas

En CSS hay 5 familias genéricas:

  1. Fuentes con serif tienen un pequeño trazo en las puntas de cada letra
  2. Fuentes san-serif. No tienen los trazos de las anteriores y tienen un aspecto minimalista y moderno.
  3. Monospace. Todas las letras tienen la misma anchura. Crean un aspecto mecánico. La fuente que vemos en nuestro código es de este tipo.
  4. Cursivas. Imitan la caligrafía humana.
  5. Fantasy. Tienen un aspecto decorativo/jugueton

Estilos de fuente

font-style

Esta propiedad se usa para darle cursiva a un texto. Tiene tres propiedades:

Este es un párrafo con estilo normal.

Este es un párrafo con estilo "italic" o en cursiva.

Este es un párrafo con estilo "oblique" u oblicuo, que es muy similar a "italic" pero con menor soporte.

font-weight

Este es un párrafo normal.

Este es un párrafo ligero.

Este es un párrafo grueso.

Este es un párrafo más grueso.

font-variant

Con esta propiedad podemos especificar que un texto se muestre en "small-caps".

Con esta fuente, todas las minúsculas de un texto se convertirán en mayúsculas pero de menor tamaño.

Tamaño de fuente

La propiedad font-size especifica el tamaño de un texto.

Ser capaz de modificar los tamaños de los diferentes textos de una web es muy importante. Sin embargo, no uses párrafos con tamños mayores para que parezcan títulos y viceversa. Usa las correspondientes etiquetas h1-h6 para títulos o p para párrafos según el caso.

Los valores de tamaño pueden ser absolutos o relativos:

Valor absoluto:

  • Establece el texto a un tamaño específico.
  • No permite que el usuario cambie el tamaño del texto desde su navegador (malo por razones de accesibilidad)
  • Un valor absoluto es útil cuando se conoce el tamaño físico del dispoitivo con el que se visualizará nuestra página.

Valor relativo

  • Establece el tamaño relativo a los elementos de alrededor.
  • Permiten al usuario cambiar el tamaño del texto en el navegador.

Valores em

Para permitir que le usuario pueda redimensionar el texto los devs o desarolladores utilizamos em en lugar de píxeles.

1em equivale al tamaño por defecto del texto que son 16 px.

Para calcular la equivalencia de píxelsses a em usa esta fórmula px/16=em
