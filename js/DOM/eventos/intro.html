Eventos en JavaSCript

Ejemplos de eventos:

  • Cuando el usuario clica en un botón (onclick).
  • Cuando el usuario pasa el cursor sobre elementos (mouseover)
  • Cuando el cursor sale de un elemento (mouseout).
  • Cuando un elemento (la web, una img) termina de cargar (onload).
  • Cuando el usuario pulsa una tecla (keydown) o la suelta (keyup).
  • Cuando un input es alterado(onchange).
  • Cuando se envía un formulario (onsubmit)
