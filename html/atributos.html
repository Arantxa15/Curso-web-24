Atributos en elementos de HTML

Todos los elementos pueden cometer atributos, los atributos proveen información adicional sobre los elementos.

Los atributos se especifican en la etiqueta de apertura. Se componen de dos partes, nombre y valor.

name="value"

href=

Es el atributo básico de los links (a). Las (a) define un hipervínculo, el atributo href define la URL de la página donde nos va a llevar el hipervínculo.

Página de búsqueda

src

La etiqueta o elemento (img) se usa para insertar una imagen en una página HTML. El atributo src indica el camino o dirección de la imagen a mostrar.

src con URL o vínculo absoluto.

src con URL o vínculo relativo

width y height

Los atributos width, ancho y height, altura, define el tamaño que queremos otorgar a la imagen en nuestra página

alt

El atributo alt define el texto alternativo para una imagen. Este texto te mostrará si por alguna razón, la imagen no puede cargarse.

style

Este atributo se usa para añadir estilos visuales a cualquier elemento. Como por ejemplo, color, fuente-font-, tamaño-size-, etc.

Clic aquí para más detalles sobre style

title

Este artibuto se usa para proveer informacion extra sobre un atributo.

Esta infromación se muestra en forma de tooltip al dejar el cursor sobre este elemento.
